How to Play

Controls

Press the Start Game button to launch the ball.

Use the left and right arrow keys on the screen or on your keyboard to move the paddle.

Use the paddle to direct the ball towards the blocks.

How to Win

Break all the blocks to win!

Use the Reset Game button to restart the game if you lose the ball.

Make sure the ball doesn't hit the bottom or it's Game Over!
